ⓘ Help & Setup Instructions

Part 1 — Before You Begin: Getting Your API Key

What is an API key and why do I need one?
This tool uses an AI engine to read your chart images and extract all the planetary positions automatically. That AI engine is provided by Anthropic — the company that created it. To use their AI, you need a personal password called an API key. Anthropic's website generates this key for you automatically when you create a free account. The AI itself does not generate the key — Anthropic's website does. You only need to do this once.

Step-by-step:

  1. Go to console.anthropic.com in your web browser
  2. Click Sign Up and create a free account with your email address and a password
  3. Once logged in, look in the left-side menu for Billing — click it
  4. Add a credit card and make a minimum deposit of $5.00. This activates your account. You will not be charged anything beyond what you deposit.
  5. Look in the left-side menu for API Keys — click it
  6. Click Create Key and give it any name you like — for example "Sahama Tool"
  7. Your key will appear on screen. It is a long string of characters that looks like this:
    sk-ant-api03-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
  8. Copy it immediately — Anthropic only shows it once. Paste it somewhere safe like a notes app on your computer.
  9. Keep your key private — treat it like a password and do not share it with others.

If you ever lose your key or think someone else has seen it, go back to console.anthropic.com, delete the old key, and create a new one.

Part 2 — What Does It Cost?

Using this tool is not free, but it is extremely inexpensive.

  • Each chart calculation costs roughly one cent or less
  • Your initial $5.00 deposit will last most students years of regular use
  • There is no monthly fee and no subscription — you only spend money when you actually use the tool

To put it in perspective: doing one chart per week for an entire year would cost approximately 50 cents.

Part 3 — How to Use This Tool

  1. Paste your API key. The first section on this page is labeled API Key. Click inside the white box and paste your key. You must do this every time you use the tool — it does not save your key between sessions. This is for your security.
  2. Screenshot your Astronomical Data table. In Shri Jyoti Star (or Jagannatha Hora, Parashara Light, or similar), take a screenshot of the Astronomical Data table. On a Mac use Command+Shift+4 to select just that part of the screen — it saves automatically to your Desktop.
  3. Upload it in Step 1. Click Choose File in Step 1 and select your screenshot. The tool reads it automatically and extracts all planetary longitudes. The extracted values appear on screen — review them and correct any if needed.
  4. Screenshot your House Cusps table. Take a screenshot of the House Cusps (Chalit/Śhrī Pati) table the same way.
  5. Upload it in Step 2. Click Choose File in Step 2 and select that screenshot. The tool reads all 12 Bhāva Madhyas from the Centre column automatically.
  6. Select D or N. In Step 3, select whether this is a Diurnal (D) or Nocturnal (N) chart.
  7. Click Calculate. Press Calculate All 51 Sahamas. All 51 sahamas appear instantly showing the name, formula, result in rāśi and degrees, and whether saikatham was applied.
  8. Print or save. Use the Print / Save PDF button to print the results or save as a PDF.
