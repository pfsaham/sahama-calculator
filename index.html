ⓘ Setup Guide

Part 1 — Getting Your API Key

Good news — you only need to do this once. After your account is set up, you simply paste your key and go.

What is an API key? This tool uses an AI engine provided by Anthropic to read your chart images and extract all planetary positions automatically. To use their AI you need a personal password called an API key. Anthropic's website generates this key for you automatically when you create a free account.

Step-by-step:

  1. Go to console.anthropic.com in your web browser
  2. Click Sign Up and create a free account with your email and a password
  3. Once logged in, click Billing in the left menu
  4. Add a credit card and make a minimum deposit of $5.00 to activate your account
  5. Click API Keys in the left menu
  6. Click Create Key — give it any name, for example "Sahama Tool"
  7. Your key appears on screen — a long string like:
    sk-ant-api03-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
  8. Copy it immediately — Anthropic only shows it once. Save it somewhere safe.
  9. Keep it private — treat it like a password.

If you ever lose your key go back to console.anthropic.com, delete it, and create a new one.

Part 2 — What Does It Cost?

Using this tool is not free, but it is extremely inexpensive.

  • Each chart calculation costs roughly one cent or less
  • Your initial $5.00 deposit will last most students years of regular use
  • There is no monthly fee and no subscription — you only spend money when you actually use the tool

To put it in perspective: doing one chart per week for an entire year would cost approximately 50 cents.

Part 3 — Tips for Best Results

  • Take screenshots using Command+Shift+4 on a Mac to select just the table area
  • Black and white screenshots give better extraction accuracy than multicolour
  • Make sure your screenshot is sharp and clear — blurry images lead to misreads
  • After extraction always scan the minutes column of each value against your original table — misreads most commonly affect the minutes
  • All extracted fields are editable — simply click and correct any value before calculating
  • D (Diurnal) is selected by default — only click N if your chart is nocturnal
  • Use the Print / Save PDF button to save your results
